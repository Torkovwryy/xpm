# XPM (eXtensible Package Manager)

> The scalable, deterministic, and secure package manager for the modern web and beyond.

![TypeScript](https://img.shields.io/badge/TypeScript-Ready-blue?style=flat-square)

XPM is not just another package manager. It is designed from the ground up with **Supply Chain Security**, **O(1) topological graph resolution**, and **Extensibility** as first-class citizens.

## 🎯 Vision

To provide the most secure, fast, and extensible foundation for dependency management across monorepos and multi-language ecosystems.

## ✨ Core Principles

- **Zero Phantom Dependencies**: Strict peer-dependency enforcement.
- **Secure by Design**: Integrity verification, post-install script isolation, and signature validations.
- **Extensible**: Hook-based architecture for custom resolution and network fetching algorithms.
- **Monorepo First**: Native workspace topology resolution.
